# E-Commerce Website using Django

## Overview
This project is an e-commerce web application built using the Django framework. It allows users to browse products, view details, add items to cart, and simulate a checkout process.

## Objective
To develop a scalable and user-friendly e-commerce platform using Django and understand the implementation of web application architecture.

## Key Features
- Product listing and product detail pages
- Add to cart functionality
- Basic checkout workflow
- Dynamic content rendering using Django templates

## Tech Stack
- Python
- Django (MVT Architecture)
- HTML, CSS
- SQLite (Database)

## System Design
The application follows Django's Model-View-Template (MVT) architecture:
- **Models**: Handle database structure (products, users, orders)
- **Views**: Process user requests and business logic
- **Templates**: Render dynamic HTML pages

## Functionality
- Displays product catalog
- Shows detailed product information
- Manages user cart
- Simulates checkout process

## Learnings
- Understanding of Django framework and MVT architecture
- Handling user requests using views
- Dynamic content rendering using templates
- Basics of building scalable web applications
